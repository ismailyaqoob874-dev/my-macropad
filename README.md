# My 4-Key Hackpad

This is my custom 4-key macropad for Hack Club Blueprint.
It uses a XIAO RP2040 and 4 switches.
Front silkscreen has my name on it.

## Project Files
- pcb/      → KiCad files + gerbers
- case/     → Fusion 360 files + plate DXF
- firmware/ → main.py (KMK firmware)
